## Requirements
- swift-tools-version: 5.8
  
## Setup
- [x] Go on [Petfinder](https://petfinder.com) to get credentials
- [x] Open Pet finder project and paste **CLIENT_ID** & **CLIENT_SECRET** token in `NetworkConstantst.swift` (`Networking` package)

- SwiftUI
- MVVM
- Combine
